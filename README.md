# Amlesh Thakur Public Research Profile Website

This package is ready for GitHub Pages public hosting.

## Public website URL after upload

https://amleshthakur00.github.io/

## Files

- `index.html` — main public profile page
- `cv_amlesh_thakur.html` — public CV page
- `paper_hctnet.html` — public research manuscript page
- `robots.txt` — allows search engines to crawl the website
- `sitemap.xml` — sitemap for Google Search Console
- `.nojekyll` — tells GitHub Pages to publish files directly

## Upload steps

1. Go to GitHub.
2. Create a public repository named `amleshthakur00.github.io`.
3. Upload all files from this ZIP into the root of the repository.
4. Go to Settings > Pages.
5. Select: Deploy from a branch > main > root.
6. Open: https://amleshthakur00.github.io/

## Google indexing

After the website is live:
1. Open Google Search Console.
2. Add URL prefix: https://amleshthakur00.github.io/
3. Verify ownership.
4. Submit this sitemap: https://amleshthakur00.github.io/sitemap.xml
5. Use URL Inspection and request indexing for the homepage.
